# 5명의 사람 이름을 입력 받아서 ArrayList에 저장한 후에 이들 중 사용자가 원하는 성씨를 가진 사람을 모두 출력하시오.

## 구현 순서
- [x] 5명의 사람 이름을 입력 받는다.
- [x] 김씨성을 가진사람을 따로 골라낸다.
- [x] 김씨성을 가진 사람을 출력한다.
- [x] 사용자에게 골라줄 성씨를 직접 입력받는다.

## 추가된 요구 사항
- 김씨만 골라주는 것 보다는 골라줄 성씨를 직접 입력하면 어떨까?
